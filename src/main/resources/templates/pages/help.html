도움말

• 설명: MetaGen 사용에 대한 가이드와 지원 정보를 제공하는 화면입니다.

• 기능:

  • 기능별 튜토리얼
    • 각 메뉴별 사용 방법 및 예시 제공
    • 동영상 튜토리얼 링크 또는 GIF 활용 가능
  • FAQ
    • 자주 묻는 질문과 답변 제공
    • 검색 기능 추가로 원하는 답변을 빠르게 찾을 수 있음
  • 고객 지원
    • 관리자 또는 지원팀에 문의할 수 있는 연락처 또는 이메일 제공
  • 릴리즈 노트
    • 최신 업데이트 정보 및 주요 변경사항 표시

• 추가 요구사항:

  • 직관적이고 간단한 UI로 구성
  • 사용자가 도움말에서 빠르게 필요한 정보를 찾을 수 있도록 검색 기능 제공
  • 다국어 지원(한국어, 영어 등) 옵션 제공
