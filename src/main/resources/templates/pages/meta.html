2. 메타데이터

• 설명: 사용자가 메타데이터를 생성 및 관리할 수 있는 화면.

• 기능:

  • 새로운 메타데이터 생성 및 수정
  • 관리자 승인 시스템
    • 승인 대기 상태 표시
    • 관리자가 승인/거절 처리 가능
    • 거절 시 피드백 제공
  • 메타데이터 필터 및 검색 기능 (이름, 생성일, 승인 상태 등)
  • 메타데이터 버전 관리 (변경 이력 추적 및 복원 기능)
  • 엑셀 템플릿 기반 데이터 업로드
    • 템플릿 다운로드 버튼 제공
    • 엑셀 파일 업로드를 통해 대량 데이터 추가
    • POI를 이용한 데이터 파싱 및 저장
    • 유효성 검사: 누락된 값, 데이터 타입 불일치, 중복 여부 확인
    • 오류 발생 시 상세 메시지 제공

논의 사항

  • **템플릿 필드 정의**: 템플릿에 포함될 필수 데이터 필드와 유효성 규칙을 명확히 정의.
  • **업로드 처리 방식**:
    • 실시간 처리: 업로드 즉시 데이터를 파싱하고 유효성 검사 수행.
    • 배치 처리: 대량 데이터를 효율적으로 처리하기 위해 비동기 배치 방식 고려.
  • **에러 관리 및 성능**:
    • 대량 데이터 업로드 시의 성능 최적화 방안.
    • 유효성 검사에서 발견된 오류 항목 표시 방식.
  • **관리자 승인 워크플로우**:
    • 승인 및 거절 프로세스의 세부 흐름 정의.
    • 거절 시 사용자 피드백 방식.
  • **배치 작업 고려**:
    • **배치 스케줄링**: 업로드 데이터를 주기적으로 처리할 배치 작업 추가.
    • **Spring Batch 활용**: 배치 작업을 Spring Batch로 구현하고, 상태 추적 및 에러 관리 추가.
    • **대량 데이터 처리**: 대용량 데이터 처리 중 메모리 최적화를 위해 Chunk 기반 로직 적용.
