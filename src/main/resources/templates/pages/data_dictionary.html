1. 데이터 사전

• 설명: 데이터베이스 테이블과 컬럼 구조를 정의하고 관리할 수 있는 화면. 엑셀 파일을 통해 데이터 사전을 업로드하거나 다운로드하여 대량의 데이터를 효율적으로 관리할 수 있습니다.

• 기능:

  • 테이블 및 컬럼 정의
    • 테이블 이름, 설명 추가
    • 컬럼 이름, 데이터 타입, 길이, 제약 조건 추가
  • 명명 규칙 관리
    • 축약어 규칙 (예: "Global → glb")
    • 컬럼 이름 컨벤션 관리
  • 참조 관계 시각화
    • 테이블 간 관계 시각화 (ERD 생성)
    • 외래 키 관계 표시
  • 설계 문서 출력
    • PDF 또는 Excel로 설계 문서 다운로드
  • 엑셀 업로드 및 다운로드
    • 엑셀 템플릿을 다운로드하여 데이터 사전을 대량 작성
    • 작성된 엑셀 파일을 업로드하여 데이터 사전 자동 생성 및 업데이트
    • 엑셀 업로드 시 필수값 검증, 데이터 타입 일치 여부 등 유효성 검사
  • 검색 및 필터링
    • 테이블 이름, 컬럼 이름, 데이터 타입 기준 검색
    • 특정 키워드로 필터링
  • 데이터베이스 역설계 : 설계를 거치지 않았거나, 관리 부실로 인해 정보가 사라진 관리안되는 프로젝트의 경우 대비
    • 기존 데이터베이스의 테이블과 컬럼 정보에서 데이터 사전 자동 생성
    • 컬럼 주석(comment) 및 데이터 타입 정보를 CSV 형식으로 업로드하여 역설계된 데이터 사전을 엑셀로 다운로드 하거나 바로 서비스에 업데이트 할 수 있는 기능 제공

논의 사항

  • **명명 규칙 자동화**:
    • 축약어 규칙을 자동으로 컬럼 이름에 적용
    • 일관성 검사 기능 추가
  • **데이터베이스 연동**:
    • 실제 데이터베이스 스키마와 동기화
    • 데이터베이스 변경사항 추적 및 업데이트
  • **ERD 생성 방식**:
    • ERD 시각화를 위한 라이브러리 선택 (예: PlantUML, Mermaid.js)
    • 테이블 간 관계를 직관적으로 표현
  • **엑셀 업로드/다운로드 추가 사항**:
    • 역설계 데이터를 수정한 엑셀 파일 업로드를 통한 업데이트 기능
    • 엑셀 업로드 시 유효성 검사: 필수값 누락, 데이터 타입 불일치 등
    • 다운로드 시 읽기 쉽게 정렬 및 스타일 적용
